## 오늘 한 것

- 블로그에 Daily 섹션 추가 (콘텐츠 컬렉션 `daily`)
- GitHub Pages 첫 배포 완료

## 배운 것

Astro 콘텐츠 컬렉션은 `src/content.config.ts`에 컬렉션을 정의하고,
`src/content/<컬렉션명>/`에 md 파일을 넣으면 자동으로 타입 안전하게 로드된다.

```ts
const daily = defineCollection({
  loader: glob({ pattern: "**/*.{md,mdx}", base: "./src/content/daily" }),
  schema: z.object({
    title: z.string(),
    date: z.coerce.date(),
    tags: z.array(z.string()).default([]),
  }),
});
```

앞으로 이 폴더에 `YYYY-MM-DD.md` 형식으로 기록을 쌓으면 된다.
